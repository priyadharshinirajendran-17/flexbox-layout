# Portfolio Website – Web Development Tasks

## Overview
This project is a simple multi-page portfolio website created as part of my web development internship tasks. It demonstrates the use of HTML, CSS, and Flexbox with a focus on clean layout, responsive design, and reusable styling.

## Pages Included
- **Home** – Introduction with profile photo and navigation
- **About** – Short description about me and my interest in web development
- **Skills** – List of technical skills displayed using Flexbox cards
- **Contact** – Basic contact information

## What I Implemented
- External CSS file for consistent styling across all pages
- Flexbox for layout and alignment
- Responsive design using media queries
- CSS Box Model (margin, padding, border, content)
- Reusable classes instead of inline styles
- Clean and professional UI with a profile photo

## Technologies Used
- HTML5
- CSS3
- Flexbox
- VS Code
- Git & GitHub

## Learning Outcome
Through this project, I gained hands-on experience in:
- Structuring multi-page websites
- Styling layouts using the CSS Box Model
- Making designs responsive for mobile and desktop
- Maintaining clean and readable code
